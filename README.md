# Projeto de Banco de Dados - Locadora

## Descrição
Projeto desenvolvido para praticar modelagem de banco de dados utilizando MySQL Workbench.

## Tecnologias Utilizadas
- MySQL
- MySQL Workbench
- GitHub

## Estrutura do Projeto
- Modelo relacional da locadora
- Script SQL para criação das tabelas
- Dados de teste para inserção no banco

## Objetivo
Praticar conceitos de modelagem de dados, relacionamentos entre tabelas, chaves primárias e estrangeiras.
